# Magictools Resources

Source: https://github.com/ellisonleao/magictools

`magictools` is a curated game-development resource list. It does not contain Codex `SKILL.md` files or plugin manifests.

Relevant picks for Spanish Bread:

- 3D assets: Poly Pizza, Quaternius, Sketchfab
- UI/icons: Games-Icons, Iconmonstr, Kenney Assets
- Textures: TextureHaven, TextureKing, Matcaps
- Level tools: Tiled, LDtk, Sprite Fusion
- Audio resources: the Audio section in `magictools` for music and audio editors

Use these as source candidates for future visual/audio upgrades, with license checks before importing assets.
